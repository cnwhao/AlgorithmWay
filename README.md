### Step1.数组
* 存储方式
> * 数组由于是紧凑连续存储,可以随机访问。
> * 插入和删除，每次必须搬移后面的所有数据以保持连续，时间复杂度 O(N)。    
> * 连续存储，内存空间必须一次性分配够，所以说数组如果要扩容，需要重新分配一块更大的空间，再把数据全部复制过去，时间复杂度 O(N)；
